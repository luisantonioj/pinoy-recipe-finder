# Pinoy Recipe Finder

A Vite + React + Tailwind single-page app that lets you browse classic Filipino recipes, view details, search, and save favorites with persistence via `localStorage`.

## Quickstart
```bash
npm i
npm run dev
```

## Tech
- React + Vite
- Tailwind CSS
- React Router v6
- Context API for Favorites (persisted to localStorage)
- Accessible components and keyboard-friendly UI
- Optional PWA install prompt via `usePWA` hook
